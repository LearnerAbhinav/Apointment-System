# AI Appointment Booking System (Production-Grade)

A complete, production-ready AI-powered appointment booking system using React, Node.js, MongoDB, and Gemini.

## 🌟 Overview
This system replaces traditional booking forms with a natural language chat interface. It extracts data using AI but enforces all business logic, validation, and conflict detection on the backend.

## 🚀 Key Features
- **AI Intent Extraction**: Gemini-powered normalization of dates and intents.
- **Interpretation Step**: Users confirm what the AI understood before booking.
- **Conflict Management**: Database-level indexing and smart slot suggestions.
- **Admin Dashboard**: Comprehensive management of all bookings.
- **Premium UI**: Glassmorphic React frontend with smooth animations.

## 📁 Structure
- `/server`: Express.js backend (MVC).
- `/client`: React (Vite) frontend.

## 🛠️ Quick Start
1. **API Key**: Get a Google Gemini API Key.
2. **Env**: Set up `.env` in the server folder.
3. **Install**: Run `npm install` in both `client` and `server`.
4. **Run**: 
   - Backend: `node src/index.js` (or `npm start`)
   - Frontend: `npm run dev`

## 💎 Design System
Built using Vanilla CSS with high-end glassmorphism, Inter/Outfit typography, and a curated slate-indigo color palette.
